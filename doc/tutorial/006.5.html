9. Changing the Timesteps/Interval

Edit the file inputs/nalu-wind/nonIsoEdgeOpenJet/nonIsoEdgeOpenJet.yaml. The following variables control the number of timesteps and the frequency of output. Changing them will alter the dataset that is created:
  • termination_time controls the end timestep for the simulation.
    • Change this value to 10
  • time_step controls the time interval for the timesteps.
    • Leave this variable alone for now (you can change it later to see what happens).
