9. Changing the Timesteps/Interval

Edit the file in inputs/cloverleaf/clover.in. The following variables control the number of timesteps and the frequency of output. Changing them will alter the dataset that is created:
  • end_step controls the end timestep ID for the simulation.
    • Change this value to 100
  • visit_frequency controls the interval at which the Cinema output will be written.
    • Do not change this value now, but you can experiment with it later.

 ...
 ymax=10.0
 zmax=10.0
 initial_timestep=0.04
 max_timestep=0.04
 end_step=50            (change this value to 100)
 test_problem 1

 visit_frequency=10
 ...
