1. Tutorial Overview

The goals of this tutorial are to:
  • Provide an overall understanding of the Pantheon project.
  • Show a workflow as designed to run on Summit.
  • Provide hands-on examples of a workflow adapted to run in the tutorial VM.
    • During this we will run the workflow, change files, and re-run the workflow. This will demonstrate the different steps, and how easy it is to experiment with a Pantheon workflow.

Tutorial Instructions

The tutorial is organized as a set of web pages with code examples and commands that are to be pasted into the shell as you work. Code snippets look like this: 

   # this is some code
   a = b
Things that are to be cut and pasted into the shell look like this: 

    this is to be cut and pasted into the shell
Please note that copy and paste are different in Firefox and the shell:
 
  copy paste
Firefox Ctrl+C Ctrl+V
Shell Shift+Ctrl+C Shift+Ctrl+V
