4. Cinema Workflow Example

The Cinema Workflow example is adapted to work on this virtual machine, but first let's look at how a workflow would run on a target HPC system. 

    cd ~/pantheon/tutorial
  • Take a look at the contents of the directory. A Pantheon workflow is designed to be a readable set of scripts and directories. The place to start is the execute script, which has the steps that the workflow will execute. 

    bootstrap.env    - variables that set up the shell env
    doc/             - documentation directory
    execute          - main execution script 
    inputs/          - a directory of inputs for the workflow 
    license.md       - license file 
    pantheon/        - pantheon data directory 
    postprocess      - postprocessing directory 
    readme.md        - basic instructions 
    run/             - run directory 
    sbang.sh         - a portable shell script 
    setup/           - setup directory
    validate/        - validate directory
