7. Virtual Machine Example

The example for this tutorial has been altered to run on the tutorial VM, but it is adapted from an HPC example. The next thing we will do is run the workflow, which will:
  • Create a work directory
  • Copy files into the work directory
  • Run the simulation, creating a Cinema database in-situ, according to several files in the workflow.
Execute the workflow, and watch the shell for output during the run: 

    cd ~/pantheon/tutorial
    ./execute
After the simulation is done, you can look at the results: 

    pushd ~/pantheon/work/pantheon/E4S/Ascent_Miniapp_Workflow/results
    firefox cinema_databases/cinema_view.html 
    popd
