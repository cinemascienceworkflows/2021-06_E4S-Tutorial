3. Running the tutorial examples

Using this tutorial, we will run a Pantheon workflow that runs a mini scientific application (cloverleaf3d), creates a Cinema database in-situ (while the simulation is running), through the Ascent infrastructure, as shown in the diagram of the project's template below.
 

Set up tutorial environment

Run the following command. This sets up a few things in the VM environment specific to this tutorial. 

    source ~/pantheon/setup-jupyter.sh
    jupyter trust ~/pantheon/data/full/cinema.ipynb

Example results

These examples show what the results will look like in a jupyter notebook: 

    jupyter notebook ~/pantheon/data/full/cinema.ipynb
This is what it looks like using the Cinema browser viewer: 

    firefox ~/pantheon/data/full/cinema_view.html
