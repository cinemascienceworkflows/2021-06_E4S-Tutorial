3. Running the tutorial examples

Using this tutorial, we will run a Pantheon workflow that runs the ExaWind/nalu-wind application (ExaWind/nalu-wind), creates a Cinema database in-situ (while the simulation is running), through the Ascent infrastructure, as shown in the diagram of the project's template below.
 

Example results

These examples show what the results will look like in a jupyter notebook. To run it, paste the following command in a shell. After the notebook loads, press the Run button.
 

    ~/pantheon/tutorial/pnotebook ~/pantheon/data/example/exawind/cinema.ipynb &
This is what it looks like using the Cinema browser viewer: 

    firefox ~/pantheon/data/example/exawind/cinema_view.html
